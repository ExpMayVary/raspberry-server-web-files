Indroduction

This is the tecnical document for the Leauge of legends Roller program

You can download the program here.

Starting the program

you start the program by running the exe included in the download.

Choosing your Champions

after opening the program you will be prompted to enter the Champions you are thinking of playing,

type the name or nickname that you will understand into the text field,
like Thresh "enter" after you have entered it.

keep entering names and pressing "enter" untill all the champs you want to play are entered.

type done and press "enter" this will start the rolling. champs i use this on:

  • Thresh
  • Lulu
  • Bard
  • Nami
  • Pyke
getting your champ

after you enter done into the program it will randomly select one of the champs you entered into it,
displaying the champs name in the program, Example:
We have Randomly selected Thresh for you to play
no champ or name is favoured in any way.

after the game

after your game is done. and if you want to play another game you can use this program again,

you can type reroll and the program will take the champs you already entered and randomly select another.
if you type redo it will restart the proccess so you can enter a new list of champs to be randomly selected.
